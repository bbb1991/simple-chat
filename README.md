## Frontend
### How to run it
```
docker build -t frontend .
docker run --rm -d -p 3000:3000 frontend
```

### Used technologies
* JavaScript
* Express
* Node.js
* yarn/npm
* socket.io

## Backend
### How to run it
```
docker build -t backend .
docker run --rm -d -p 5000:5000 backend
```

### Used technologies
* python3
* flask
* socket.io
